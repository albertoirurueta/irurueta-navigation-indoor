## Usage

Add the following dependency to your project:

Latest release:
```

    <groupId>com.irurueta</groupId>
    <artifactId>irurueta-navigation-indoor</artifactId>
    <version>1.0.2</version>
    <scope>compile</scope>
</dependency>
```
